# 前端组件库

## 概述

前端组件库包含聊天界面、图表可视化、布局、通用组件等可复用 UI 组件，构建于 React 19 + TypeScript + Tailwind CSS + ECharts 之上。

## 聊天组件

定义在 `frontend/src/components/chat/` 目录：

| 组件 | 文件 | 行数 | 功能 |
|------|------|------|------|
| `AgentAvatar` | `AgentAvatar.tsx` | 全局 | Agent 头像组件 |
| `WelcomeScreen` | `WelcomeScreen.tsx` | 全局 | 聊天起始欢迎页 |
| `ConversationTimeline` | `ConversationTimeline.tsx` | 全局 | 对话时间线 |
| `ThinkingTimeline` | `ThinkingTimeline.tsx` | 全局 | Agent 思维过程时间线 |
| `ToolProgressIndicator` | `ToolProgressIndicator.tsx` | 全局 | 工具执行进度指示器 |
| `ProgressBar` | `ProgressBar.tsx` | 全局 | 进度条 |
| `MessageBubble` | `MessageBubble.tsx` | 全局 | 消息气泡组件 |
| `RunnerStatus` | `RunnerStatus.tsx` | 全局 | Agent Runner 状态 |
| `SwarmStatusCard` | `SwarmStatusCard.tsx` | 全局 | Swarm 运行状态卡片 |
| `MandateProposalCard` | `MandateProposalCard.tsx` | 全局 | 交易授权提案卡片 |
| `RunCompleteCard` | `RunCompleteCard.tsx` | 全局 | 回测完成结果卡片 |
| `MetricsCard` | `MetricsCard.tsx` | 全局 | 绩效指标卡片 |
| `PineScriptViewer` | `PineScriptViewer.tsx` | 全局 | Pine Script 代码查看器 |
