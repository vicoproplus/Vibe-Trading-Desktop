# 前端页面模块

## 概述

前端页面模块是 Vibe Trading 的 React SPA 应用，基于 Vite + React 19 + TypeScript 构建，提供 Agent 交互、回测分析、因子研究、设置管理等完整功能界面。

**源文件**: `frontend/src/pages/` 和 `frontend/src/router.tsx`

## 路由体系

路由定义在 `frontend/src/router.tsx`，采用 React Router v7 的 lazy loading 方案：

| 路由 | 文件 | 功能 |
|------|------|------|
| `/` | `frontend/src/pages/Home.tsx` | 首页/仪表盘 |
| `/agent` | `frontend/src/pages/Agent.tsx` | Agent 聊天交互 |
| `/runtime` | `frontend/src/pages/Runtime.tsx` | 桌面端运行时设置 |
| `/settings` | `frontend/src/pages/Settings.tsx` | LLM/数据源设置 |
| `/runs/:runId` | `frontend/src/pages/RunDetail.tsx` | 回测运行详情 |
| `/compare` | `frontend/src/pages/Compare.tsx` | Alpha 对比 |
| `/correlation` | `frontend/src/pages/Correlation.tsx` | 相关性分析 |
| `/alpha-zoo` | `frontend/src/pages/AlphaZoo.tsx` | Alpha 因子库浏览 |
| `/reports` | `frontend/src/pages/Reports.tsx` | 影子报告列表 |
| `/login` | `frontend/src/pages/auth/Login.tsx` | 用户登录 |
| `/profile` | `frontend/src/pages/profile/Profile.tsx` | 用户资料 |

## 核心页面详解

### Agent 聊天页

`Agent.tsx` 是核心交互页面，主要功能：
- 消息输入与发送
- SSE 流式接收 Agent 回复
- 显示工具调用进度（`ToolProgressIndicator`）
- 显示思维链路（`ThinkingTimeline`）
- 回测结果卡片（`RunCompleteCard`）
- Swarm 状态卡片（`SwarmStatusCard`）
- Mandate 提案卡片（`MandateProposalCard`）

**使用场景**:
```tsx
// 访问 Agent 页面
<Route path="/agent" element={<Agent />} />
```

### 回测详情页

`RunDetail.tsx` 展示单次回测运行完整详情：
- 策略代码预览
- 绩效指标面板（`MetricsCard`）
- 权益曲线图（`EquityChart`）
- 交易记录列表
- 配置信息

### 设置页

`Settings.tsx` 管理应用配置：
- LLM 提供者选择（OpenRouter / OpenAI / DeepSeek 等 13+）
- API Key 管理
- 数据源配置
- 可选依赖管理（`OptionalDepsManager`）

### 因子库页

`AlphaZoo.tsx` 浏览所有可用 Alpha 因子：
- 按因子库分类筛选（Alpha101 / GTJA191 / Qlib158 / Academic）
- 因子详情查看
- 因子基准测试触发
