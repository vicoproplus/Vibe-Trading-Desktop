# 回测引擎模块

## 概述

回测引擎提供多市场、多策略的回测能力，支持股票、期货、外汇、加密货币等 9 个市场的策略回测，含完整的指标计算、优化器、基准对比和文中分析。

**源文件**: `agent/backtest/` 目录

## 核心架构

### 1. 回测入口

`main()` 函数定义在 `agent/backtest/runner.py` L392-L544，是回测执行的统一入口。接收 `run_dir` 参数（含配置文件的目录），流程：

1. 加载配置 → `BacktestConfigSchema` 验证
2. 检测市场 → 选择引擎（`_create_market_engine()` L547-L601）
3. 加载数据 → `_fetch_auto()` L623-L670
4. 执行回测 → `BaseEngine.run_backtest()`
5. 计算指标 → `backtest/metrics.py`
6. 写回结果 → `_write_artifacts()`

**使用场景**:
```bash
vibe-trading backtest /path/to/run_dir
```

### 2. 配置模型

`BacktestConfigSchema` Pydantic 模型（`agent/backtest/runner.py` L54-L143），核心字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| `codes` | List[str] | 交易标的代码列表 |
| `start_date` | str | 开始日期 (YYYYMMDD) |
| `end_date` | str | 结束日期 (YYYYMMDD) |
| `interval` | str | K线周期 (1D, 60min, 30min 等) |
| `engine` | str | 引擎类型 (china_a, crypto, forex 等) |
| `source` | str | 数据源 (yfinance, akshare, ccxt 等) |
| `cash` | float | 初始资金 (默认 1,000,000) |
| `commission` | float | 手续费率 (默认 0.0003) |
| `slippage` | float | 滑点 (默认 0.001) |
| `signal_engine` | str | 信号引擎代码路径 |

### 3. 引擎体系

`BaseEngine` 抽象基类定义在 `agent/backtest/engines/base.py` L255-L767，提供了回测执行的核心框架。

具体引擎实现：

| 引擎 | 文件 | 行数 | 覆盖市场 |
|------|------|------|----------|
| `ChinaAEngine` | `backtest/engines/china_a.py` | 全局 | A 股（含涨跌停限制、T+1） |
| `ChinaFuturesEngine` | `backtest/engines/china_futures.py` | 全局 | 中国期货（保证金、逐日盯市） |
| `CryptoEngine` | `backtest/engines/crypto.py` | 全局 | 加密货币（永续合约、资金费率） |
| `ForexEngine` | `backtest/engines/forex.py` | 全局 | 外汇（点差、隔夜利息） |
| `GlobalEquityEngine` | `backtest/engines/global_equity.py` | 全局 | 全球股票（融券、股息） |
| `GlobalFuturesEngine` | `backtest/engines/global_futures.py` | 全局 | 全球期货 |
| `CompositeEngine` | `backtest/engines/composite.py` | 全局 | 跨市场组合回测 |
| `OptionsPortfolioEngine` | `backtest/engines/options_portfolio.py` | 全局 | 期权组合 |

`BaseEngine` 核心方法：

- **`run_backtest()`** (L360-L498)：主循环，逐 bar 执行信号 → 交易 → 记账
- **`_execute_bars()`** (L502-L547)：遍历 OHLC 数据
- **`_rebalance()`** (L559-L634)：调仓逻辑
- **`_calc_equity()`** (L549-L557)：权益计算
- **`_close_position()`** (L636-L672)：平仓逻辑
- **`_write_artifacts()`** (L676-L751)：结果写入

### 4. 数据加载器体系

数据加载器定义在 `agent/backtest/loaders/` 目录下，支持 30+ 数据源：

| 加载器 | 文件 | 覆盖数据 |
|--------|------|----------|
| `YFinanceLoader` | `backtest/loaders/yfinance_loader.py` | 全球股票/ETF |
| `AKShareLoader` | `backtest/loaders/akshare_loader.py` | A 股/期货/宏观 |
| `CCXTLoader` | `backtest/loaders/ccxt_loader.py` | 加密货币 |
| `EastMoneyLoader` | `backtest/loaders/eastmoney_loader.py` | A 股/板块 |
| `TuShareLoader` | `backtest/loaders/tushare.py` | A 股/指数/基金/期货/期权 |
| `BaostockLoader` | `backtest/loaders/baostock_loader.py` | A 股 |
| `FinnhubLoader` | `backtest/loaders/finnhub_loader.py` | 全球股票 |
| `FxLoader` | `backtest/loaders/stooq_loader.py` | 外汇 |
| `LocalLoader` | `backtest/loaders/local_loader.py` | 本地 CSV |
| `AlphaVantageLoader` | `backtest/loaders/alphavantage_loader.py` | 全球股票 |

数据加载器的注册机制在 `agent/backtest/loaders/registry.py`，通过 `registry.py` 中的映射选择适当的加载器。

### 5. 指标计算

定义在 `agent/backtest/metrics.py`，包含：

- 收益率（年化/累计）
- 夏普比率
- 最大回撤
- 波动率
- 胜率
- 盈亏比
- Calmar 比率
- Sortino 比率
- 信息比率
- Alpha / Beta

### 6. 优化器

定义在 `agent/backtest/optimizers/` 目录下：

| 优化器 | 文件 | 方法 |
|--------|------|------|
| MeanVariance | `optimizers/mean_variance.py` | 均值方差优化 |
| RiskParity | `optimizers/risk_parity.py` | 风险平价 |
| EqualVolatility | `optimizers/equal_volatility.py` | 等波动率 |
| MaxDiversification | `optimizers/max_diversification.py` | 最大分散化 |

### 7. 安全校验

回测执行前进行安全校验（`agent/backtest/runner.py` L202-L291）：

- 信号引擎代码 AST 静态分析
- 禁止 `import os`, `import subprocess`, `eval()`, `exec()` 等危险操作
- 限制类和函数定义体
- 验证引擎类接口完整性

**使用场景**:
```python
from backtest.runner import main
from pathlib import Path
main(run_dir=Path("/path/to/run_dir"))
```
