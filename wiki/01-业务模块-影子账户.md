# 影子账户模块

## 概述

影子账户模块（Shadow Account）提供完整的纸面交易模拟系统，支持从策略代码生成、回测执行到 HTML/PDF 报告生成的全流程。它模拟真实交易环境（含手续费、滑点、市场冲击），帮助用户在投入实盘前验证策略效果。

**源文件**: `agent/src/shadow_account/` 目录

## 核心组件

### 1. 策略代码生成

`codegen.py`（`agent/src/shadow_account/codegen.py`）从用户自然语言描述自动生成交易策略代码：

- 使用 LLM 生成策略 Python 代码
- 验证代码语法和安全性
- 生成 SignalEngine 类（策略引擎）
- 支持 Pine Script 转换

### 2. 回测执行

`backtester.py`（`agent/src/shadow_account/backtester.py`）执行影子回测：

- 使用同回测引擎（BaseEngine）但附加额外检查
- 记录每笔模拟交易
- 计算模拟账户权益曲线
- 记录交易决策理由

### 3. 信号扫描

`scanner.py`（`agent/src/shadow_account/scanner.py`）扫描市场生成交易信号：

- 支持多股票扫描
- 按信号引擎规则生成买卖信号
- 信号聚合和排序

### 4. 报告生成

`reporter.py`（`agent/src/shadow_account/reporter.py`）生成 HTML/PDF 报告：

- 策略绩效摘要
- 交易列表
- 权益曲线图
- 回撤分析
- 关键指标（夏普、最大回撤、胜率）
- PDF 导出（依赖 weasyprint，桌面版不内置）

### 5. 存储

`storage.py`（`agent/src/shadow_account/storage.py`）管理影子账户数据的持久化：

- 回测结果保存
- 策略版本管理
- 报告缓存

### 6. 数据模型

`models.py`（`agent/src/shadow_account/models.py`）定义影子账户的数据模型：

- `ShadowRun`：影子运行记录
- `ShadowTrade`：模拟交易
- `ShadowMetrics`：绩效指标
- `ShadowReport`：报告数据

**使用场景**:
```bash
vibe-trading shadow scan                        # 扫描生成信号
vibe-trading shadow backtest                    # 执行影子回测
vibe-trading shadow report                      # 生成报告
```

```python
from src.shadow_account.codegen import generate_strategy
from src.shadow_account.backtester import run_shadow_backtest
from src.shadow_account.reporter import render_report

code = generate_strategy("基于 MACD 金叉买入，死叉卖出")
result = run_shadow_backtest(code, symbols=["AAPL", "MSFT"])
render_report(result, format="html")  # 或 format="pdf"
```
