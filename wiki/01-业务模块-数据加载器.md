# 数据加载器模块

## 概述

数据加载器模块是 Vibe Trading 的数据接入层，提供统一的数据获取接口，屏蔽不同数据源的差异。支持 30+ 数据源，涵盖全球股票、A 股、加密货币、期货、外汇、宏观经济等各类数据。

**源文件**: `agent/backtest/loaders/` 目录

## 核心架构

### 1. 加载器注册中心

定义在 `agent/backtest/loaders/registry.py`，维护数据源与加载器的映射关系。回测执行时通过 `_get_loader()` 函数（`agent/backtest/runner.py` L357-L372）根据 `source` 字段选取对应加载器。

### 2. 基础加载器

`BaseLoader` 抽象类定义在 `agent/backtest/loaders/base.py`，提供统一接口：
- `fetch(codes, start_date, end_date, fields, interval)`：获取 OHLC 数据
- `validate_ohlc()`：OHLC 数据验证

### 4. HTTP 客户端基类

`_HTTPLoaderBase` 在 `agent/backtest/loaders/_http.py` 中定义，为需要 HTTP 请求的加载器提供统一的重试、超时、错误处理机制。

**使用场景**:
```python
from backtest.loaders.registry import get_loader
from backtest.loaders import NoAvailableSourceError

loader = get_loader("yfinance")
data = loader.fetch(
    codes=["AAPL", "MSFT"],
    start_date="20230101",
    end_date="20231231",
    interval="1D"
)
```
