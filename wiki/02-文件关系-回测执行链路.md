# 文件关系 - 回测执行链路

## 链路概述

回测执行从配置加载开始，经数据获取、引擎执行、指标计算到结果输出，是一条完整的批处理流水线。

```
用户配置 → 数据加载 → 引擎执行 → 指标计算 → 结果输出
```

## 详细链路

### 1. 入口点

```
回测有两种入口：
  A) CLI: vibe-trading backtest <run_dir>
     → agent/cli/main.py → main()
       → agent/backtest/runner.py → main(run_dir)  (L392-L544)
  
  B) API: POST /sessions/{id}/messages → Agent → backtest_tool
     → agent/src/tools/backtest_tool.py
       → agent/backtest/runner.py → main(run_dir)
```

### 2. 配置加载与验证

```
agent/backtest/runner.py
  ├── BacktestConfigSchema        (L54-L143)      # Pydantic 配置模型
  │     ├── codes                     # 标的代码列表
  │     ├── start_date / end_date     # 回测时间范围
  │     ├── interval                  # K 线周期
  │     ├── engine                    # 引擎类型
  │     ├── source                    # 数据源
  │     ├── cash / commission / slippage  # 资金/费用参数
  │     └── signal_engine             # 信号引擎代码路径
  │
  └── _validate_signal_engine_source()  (L244-L271)  # 安全验证
```

### 3. 数据加载

```
agent/backtest/runner.py
  ├── _detect_source()             (L312-L322)    # 自动检测数据源
  ├── _get_loader()                (L357-L372)    # 获取加载器实例
  └── _fetch_auto()                (L623-L670)    # 加载数据
        │
        ▼
agent/backtest/loaders/
  ├── registry.py                              # 加载器注册中心

        │

```

### 4. 引擎执行

```
agent/backtest/runner.py
  └── _create_market_engine()      (L547-L601)     # 选择引擎
        │
        ▼
agent/backtest/engines/
  ├── base.py → BaseEngine.run_backtest()  (L360-L498)
  │     └── _execute_bars()                (L502-L547)
  │           ├── on_bar()                 (L330-L334)  # 信号处理
  │           ├── _rebalance()             (L559-L634)  # 调仓
  │           ├── _calc_equity()           (L549-L557)  # 权益计算
  │           └── _close_position()        (L636-L672)  # 平仓
  │
  ├── china_a.py → ChinaAEngine             # A 股市场

  ├── forex.py → ForexEngine                # 外汇
  ├── global_equity.py → GlobalEquityEngine # 全球股票
  ├── global_futures.py → GlobalFuturesEngine # 全球期货
  ├── composite.py → CompositeEngine        # 跨市场组合
  └── options_portfolio.py                  # 期权组合
```

### 5. 指标计算

```
agent/backtest/metrics.py
  ├── 年化收益率
  ├── 夏普比率
  ├── 最大回撤
  ├── 波动率
  ├── 胜率 / 盈亏比

```

```
agent/backtest/optimizers/
  ├── mean_variance.py              # 均值方差
  ├── risk_parity.py                # 风险平价
```


### 核心调用链关系

| 步骤 | 调用方 → 被调用方 | 关键代码位置 |
|------|-------------------|-------------|
| CLI 入口 | `cli/main.py:main()` → `backtest/runner.py:main()` | `runner.py:392` |
| 数据加载 | `_fetch_auto()` → `_get_loader()` → Loader.fetch() | `runner.py:623` |
| 引擎创建 | `_create_market_engine()` → `BaseEngine()` | `runner.py:547` |
| 回测循环 | `BaseEngine.run_backtest()` → `_execute_bars()` | `base.py:360` |
| 调仓 | `_execute_bars()` → `_rebalance()` | `base.py:559` |
| 安全校验 | `_validate_signal_engine_source()` | `runner.py:244` |
| 基准对比 | `BaseEngine.run_backtest()` → `resolve_benchmark()` | `base.py:444` |
