# 文件关系 - 实盘交易链路

## 链路概述

实盘交易链路从运行时调度启动，经过 Mandate 授权检查、订单门控验证、券商连接器执行，到审计日志记录，是一条安全关键路径。

```
调度器 → Mandate 授权 → 订单门控 → 券商连接器 → 审计
```

## 详细链路

### 1. 实盘运行器启动

```
API 层:
  POST /live/runner/start
  → api_server.py → start_runner_endpoint()       (L3259-L3311)
    → _build_live_runner()                         (L3145-L3243)
      → _live_broker_adapter()                     (L3108-L3142)
        │
        ▼  runner 构建
```

### 2. 运行时调度

```
agent/src/live/runtime/
  ├── scheduler.py → Scheduler                    # 定时任务调度
  ├── runner.py → LiveRunner                      # 持续运行 Agent
  ├── triggers.py → Trigger                       # 触发条件
  ├── jobstore.py → JobStore                      # 任务持久化
  ├── liveness.py → Liveness                      # 活跃性探针
  ├── reconcile.py → Reconcile                    # 订单对账
  └── flatten.py → Flatten                        # 平仓
        │
        ▼  Agent 生成交易决策
```

### 3. Mandate 授权检查

```
agent/src/live/mandate/
  ├── model.py           # Mandate 数据模型
  ├── store.py           # 持久化存储
  └── commit.py          # 提交确认
        │
        ▼
  API: POST /mandate/commit → commit_mandate_endpoint()  (L2808-L2853)
```

### 4. 订单门控 (安全闸门)

```
agent/src/live/sdk_order_gate.py → SDKOrderGate
  ├── 检查 Halt 状态 (`agent/src/live/halt.py`)
  │     └── halt_flag_set() / trip_halt() / clear_halt()
  │
  ├── 检查 Mandate 授权
  │     └── Mandate 验证
  │
  ├── 检查交易额度
  │     └── enforcement.py → Enforcement
  │
  └── 记录审计
        └── audit.py → write_live_action()
```

API 入口：
- `POST /live/halt` → `halt_live_endpoint()` (L2856-L2878)
- `POST /live/resume` → `resume_live_endpoint()` (L2881-L2902)
- `GET /live/status` → `live_status_endpoint()` (L3013-L3051)

### 5. 券商连接器执行

```
agent/src/trading/service.py
  └── connector_profile_id_for_broker()
        │
        ▼
agent/src/trading/connectors/
  ├── alpaca/sdk.py       # Alpaca (美股)
  ├── binance/sdk.py      # Binance (加密货币)
  ├── dhan/sdk.py         # Dhan (印度)
  ├── futu/sdk.py         # 富途 (港股/美股/A股)
  ├── ibkr/sdk.py         # Interactive Brokers (全球)
  ├── longbridge/sdk.py   # Longbridge (港股/美股)
  ├── okx/sdk.py          # OKX (加密货币)
  ├── robinhood/sdk.py    # Robinhood (美股)
  ├── shoonya/sdk.py      # Shoonya (印度)
  └── tiger/sdk.py        # 老虎 (港股/美股/A股)
        │
        ▼
  券商 API (HTTP/WebSocket)
```

### 6. 审计记录

```
agent/src/live/audit.py → write_live_action()
  └── 记录存储到 ~/.vibe-trading/live/<broker>/audit/

agent/src/live/classification.py     # 交易分类
agent/src/live/daily_count.py        # 日交易计数
agent/src/live/paths.py              # 路径管理
agent/src/live/registry.py           # 运行时注册
```

### 安全关键调用链关系

| 步骤 | 调用方 → 被调用方 | 关键文件 |
|------|-------------------|----------|
| 授权确认 | API → Mandate Store | `agent/src/live/mandate/store.py` |
| 熔断检查 | SDKOrderGate → Halt | `agent/src/live/halt.py` |
| 订单检查 | SDKOrderGate → Enforcement | `agent/src/live/enforcement.py` |
| 交易执行 | LiveRunner → SDK | `agent/src/live/runtime/runner.py` |
| 审计记录 | 各环节 → Audit | `agent/src/live/audit.py` |
| 对账 | LiveRunner → Reconcile | `agent/src/live/runtime/reconcile.py` |
