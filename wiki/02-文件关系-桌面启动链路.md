# 文件关系 - 桌面启动链路

## 链路概述

桌面应用的启动流程从 Tauri 主入口开始，经历资源解析、端口选择、Python 侧车启动、健康检查，最终呈现完整的 Webview 界面。

```
main.rs → 资源解析 → 运行时目录 → 端口选择 → 侧车启动 → 健康检查 → Webview 导航
```

## 详细链路

### 1. Tauri 主入口

```
src-tauri/src/main.rs
  ├── Tauri Builder 初始化
  ├── ResourcesResolver 解析嵌入式资源
  ├── PortPicker 选择可用端口
  ├── Sidecar 启动 Python 进程
  └── Webview 窗口管理
```

### 2. 资源解析

```
src-tauri/src/resources.rs → ResourcesResolver
  ├── 解析 bundle 中嵌入的资源路径
  │     - .desktop-build/python-runtime/  (Python 3.12 运行时)
  │     - .desktop-build/agent/           (Agent 代码)
  │     - frontend/dist/                  (前端 SPA)
  │
  └── 首次运行 / 版本升级时复制 Agent 代码
        └── 目标: ~/.vibe-trading/runtime/agent/
```

### 3. 运行时目录管理

```
src-tauri/src/runtime_dir.rs
  ├── ~/.vibe-trading/runtime/ 目录初始化
  ├── 子目录：agent/, logs/, libs/
  └── VERSION 文件读写
```

### 4. 版本管理

```
src-tauri/src/version.rs
  ├── 读取 bundle 内嵌版本号
  ├── 读取 ~/.vibe-trading/runtime/VERSION
  └── 比较版本 → 需要升级时触发复制
```

### 5. 端口选择

```
src-tauri/src/port.rs → PortPicker
  ├── 绑定 127.0.0.1:0 (随机端口)
  ├── 确保端口可用
  └── 返回端口号
```

### 6. Python 侧车启动

```
src-tauri/src/sidecar.rs → SidecarManager
  ├── 环境变量设置
  │     ├── PYTHONPATH → site-packages 路径
  │     └── VITE_TRADING_AGENT_DIR → Agent 代码路径
  │
  ├── 启动命令:
  │     python3 -c "import cli; cli.main(['serve', '--port', <port>])"
  │
  ├── 健康检查 (异步轮询 GET /health, 超时 60s)
  │
  └── 进程终止:
        └── killpg(SIGTERM) → uvicorn timeout_graceful_shutdown=5
```

### 7. 构建配置

```
src-tauri/tauri.conf.json
  ├── bundle.resources:
  │     - .desktop-build/python-runtime/**
  │     - .desktop-build/agent/**
  │     - frontend/dist/**
  │
  ├── identifier: ai.vibetrading.desktop
  │
  └── 窗口配置
```

### 8. 构建流水线

```
构建脚本 (scripts/desktop/):
  ├── fetch-runtime.sh / fetch-runtime.ps1    # 下载嵌入式 Python 3.12
  ├── install-deps.sh / install-deps.ps1     # pip 安装依赖
  ├── assemble.sh / assemble.ps1             # 组装到 .desktop-build/
  └── build-dmg.sh / build-windows.ps1       # cargo tauri build

构建产物:
  ├── macOS: Vibe Trading.app / .dmg
  └── Windows: Vibe Trading.msi
```

### 核心调用链关系

| 步骤 | 调用方 → 被调用方 | 关键文件 |
|------|-------------------|----------|
| Tauri 入口 | `main.rs` → Tauri Builder | `src-tauri/src/main.rs` |
| 资源解析 | `ResourcesResolver::resolve()` | `src-tauri/src/resources.rs` |
| 目录管理 | `RuntimeDir::setup()` | `src-tauri/src/runtime_dir.rs` |
| 端口选择 | `PortPicker::pick_port()` | `src-tauri/src/port.rs` |
| 侧车启动 | `SidecarManager::start()` | `src-tauri/src/sidecar.rs` |
| 健康检查 | `SidecarManager::wait_healthy()` | `src-tauri/src/sidecar.rs` |
| 版本检查 | `Version::check()` | `src-tauri/src/version.rs` |
| 单元测试 | `cargo test` | `src-tauri/tests/sidecar_tests.rs` |
